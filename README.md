# рҹ“– Overview

**AttendX** is a modern cloud-based Attendance Management System designed to simplify attendance tracking for teachers and students.

It provides a fast, intuitive, and responsive interface that enables teachers to manage attendance efficiently while allowing students to check their attendance records instantly.

Built with **React**, **Firebase Authentication**, and **Cloud Firestore**, AttendX stores attendance securely in the cloud and delivers a smooth user experience across desktop and mobile devices.
